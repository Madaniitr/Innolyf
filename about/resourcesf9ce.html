Logos et ressources

Vous trouverez ici les ressources graphiques officielles de Stripe. Vous pouvez les inclure à votre page Web, à votre application mobile ou à vos documents sur papier.

Logo Stripe

Le logo bleu est idéal pour les arrière-plans clairs, le logo blanc est fait pour les arrière-plans sombres.

Téléchargez le kit graphique

Badges "Powered by Stripe"

Ces badges sont conçus pour fonctionner avec quasiment n'importe quel design. Nous vous suggérons de les relier à stripe.com.

SVG, PDF, PNG
